Development Setup

On this page

Settings ​

Configure Tafil to match your workflow.

Access Settings ​

Click the gear icon in the sidebar, or use Command Palette (⌘K → "Settings").

Available Settings ​

Default IDE ​

Choose your preferred code editor:

  • VS Code
  • Cursor
  • WebStorm
  • IntelliJ IDEA
  • Sublime Text
  • Atom

When set, clicking the IDE button opens projects directly without prompting.

Default Terminal ​

Choose your preferred terminal:

macOS:

  • Terminal.app
  • iTerm
  • Hyper
  • Warp
  • Kitty
  • Alacritty

Windows:

  • PowerShell
  • Windows Terminal
  • CMD

Linux:

  • GNOME Terminal
  • Konsole
  • Alacritty
  • Kitty

Theme ​

Toggle between Dark and Light mode.

Settings Storage ​

Settings are stored locally:

PlatformLocation
macOS~/Library/Application Support/Tafil/
Windows%APPDATA%\Tafil\
Linux~/.config/Tafil/

Reset Settings ​

To reset all settings:

  1. Quit Tafil
  2. Delete the settings file at the location above
  3. Restart Tafil
