# Keyboard Shortcuts

Quick reference for all keyboard shortcuts.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘K` / `Ctrl+K` | Open Command Palette |
| `⌘R` / `Ctrl+R` | Refresh Projects |
| `Esc` | Close modal / palette |

## Command Palette Navigation

| Key | Action |
|-----|--------|
| `↑` | Move selection up |
| `↓` | Move selection down |
| `Enter` | Execute selected command |
| `Esc` | Close palette |

## Project Cards

| Action | How |
|--------|-----|
| Run project | Click Play button |
| Stop project | Click Stop button |
| Open in IDE | Click Code button |
| Open in Terminal | Click Terminal button |
| Open in Browser | Click Browser button |
| View Insights | Click Chart button |
| Add to Collection | Click Folder button |

## Sidebar Navigation

| Action | How |
|--------|-----|
| All Projects | Click "All Projects" |
| Running | Click "Running" |
| Collection | Click collection name |
| New Collection | Click "+" button |
| Settings | Click gear icon |
