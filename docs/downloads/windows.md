# Windows Installation

## Download

| Type | File |
|------|------|
| Installer | `Tafil-x.x.x-win-x64.exe` |
| Portable | `Tafil-x.x.x-portable.exe` |

[Download from GitHub Releases →](https://github.com/Toseef-Ahmad/Tafil/releases)

## Installation (Installer Version)

1. Download the `.exe` installer
2. Run the installer
3. Follow the installation wizard
4. Launch Tafil from the Start Menu

## Installation (Portable Version)

1. Download the portable `.exe`
2. Place it in any folder (e.g., `C:\Tools\Tafil`)
3. Run the executable directly

No installation required for portable version.

## First Launch - SmartScreen Warning

Windows SmartScreen may show a warning because the app isn't signed with an EV certificate.

**To run the app:**

1. Click **More info**
2. Click **Run anyway**

This is normal for open-source applications.

## Uninstall

**Installer version:**
1. Open Settings → Apps
2. Find Tafil and click Uninstall

**Portable version:**
1. Delete the executable
2. (Optional) Delete `%APPDATA%\Tafil` folder
