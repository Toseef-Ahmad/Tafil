# Themes

Customize Tafil's appearance.

## Available Themes

| Theme | Description |
|-------|-------------|
| Dark | Default dark theme |
| Light | Light theme for bright environments |

## Change Theme

1. Open **Settings** (gear icon in sidebar)
2. Toggle **Dark Mode** on or off

Or click the **theme toggle** button in the header.

## Theme Details

### Dark Theme

- Background: `#1a1a1a`
- Cards: `#252525`
- Accent: Green (`#4EC9B0`)
- Text: White/Gray

Inspired by:
- VS Code Dark+
- Linear
- Raycast

### Light Theme

- Background: `#ffffff`
- Cards: `#f5f5f5`
- Accent: Green
- Text: Dark gray

## System Preference

Tafil respects your system's dark/light mode preference on first launch.

You can override this in Settings.

## Persistence

Your theme preference is saved and restored when you reopen Tafil.
