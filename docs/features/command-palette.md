# Command Palette

The command palette provides quick access to all Tafil features.

## Open Command Palette

| Platform | Shortcut |
|----------|----------|
| macOS | `⌘K` |
| Windows/Linux | `Ctrl+K` |

## Available Commands

### Project Actions
- **Search projects** - Type to filter by name or path
- **Run project** - Start the dev server
- **Stop project** - Stop a running dev server
- **Open in IDE** - Open in your default editor
- **Open in Terminal** - Open in your default terminal
- **Open in Browser** - Open running project in browser

### Navigation
- **All Projects** - Show all projects
- **Running** - Show running projects
- **Collections** - Jump to a collection
- **Settings** - Open settings

### Scanning
- **Scan Home** - Scan home directory
- **Scan Folder** - Scan specific folder
- **Refresh** - Refresh current view

## Keyboard Navigation

| Key | Action |
|-----|--------|
| `↑` `↓` | Navigate results |
| `Enter` | Execute selected command |
| `Esc` | Close palette |

## Tips

- **Fuzzy search:** Type partial words to find projects
- **Quick actions:** Select a project then choose an action
- **Fast switching:** Use collections for quick filtering
