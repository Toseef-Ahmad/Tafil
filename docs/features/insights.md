# Project Insights

View detailed information about any project.

## Open Insights

Click the **chart icon** (ðŸ“Š) on any project card.

## Information Displayed

### Project Details

- **Name** - From package.json
- **Location** - Full path on disk
- **Framework** - Detected framework type

### Dependencies

- **Status** - Installed or Missing
- **Count** - Number of dependencies
- **Action** - Install button if missing

### Git Information

- **Branch** - Current branch name
- **Last Commit** - Commit message
- **Commit Date** - When committed

### Running Status

- **State** - Running or Stopped
- **Port** - Active port number
- **Actions** - Start/Stop/Open

## Quick Actions

From the insights panel:

| Action | Description |
|--------|-------------|
| Run | Start dev server |
| Stop | Stop running server |
| Open in IDE | Launch in editor |
| Open in Terminal | Launch terminal |
| Install | Run npm install |
| Remove node_modules | Free disk space |

## System Metrics

The Insights view in the sidebar shows:

- Total projects scanned
- Currently running projects
- Projects with missing dependencies
- Total disk space used by node_modules
