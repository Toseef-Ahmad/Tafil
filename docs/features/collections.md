# Collections

Collections help you organize projects into groups.

## Built-in Collections

| Collection | Description |
|------------|-------------|
| All Projects | Shows all scanned projects |
| Running | Shows projects with active dev servers |
| Uncategorized | Projects not in any custom collection |

## Create a Collection

1. Click **+ New Collection** in the sidebar
2. Enter a name (e.g., "Work", "Personal", "Client Name")
3. Press Enter or click Create

## Add Projects to Collections

1. Find the project card
2. Click the **folder icon** (üìÅ)
3. Select one or more collections
4. Click Save

Projects can belong to multiple collections.

## Collection Badges

Projects show badges indicating their collections. This helps you quickly identify project groupings.

## Filter by Collection

Click a collection in the sidebar to filter the view to only those projects.

## Rename a Collection

1. Right-click the collection in the sidebar
2. Select "Rename"
3. Enter new name
4. Press Enter

## Delete a Collection

1. Right-click the collection in the sidebar
2. Select "Delete"
3. Confirm

**Note:** Deleting a collection doesn't delete the projects, only the grouping.

## Collection Ideas

- **By Client:** "Acme Corp", "Startup X"
- **By Status:** "Active", "Archived", "Maintenance"
- **By Technology:** "React Apps", "APIs", "Full Stack"
- **By Priority:** "High Priority", "Side Projects"
