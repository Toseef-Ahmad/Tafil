# IDE Integration

Open projects directly in your preferred code editor.

## Supported IDEs

| IDE | Platforms |
|-----|-----------|
| VS Code | macOS, Windows, Linux |
| Cursor | macOS, Windows, Linux |
| WebStorm | macOS, Windows, Linux |
| IntelliJ IDEA | macOS, Windows, Linux |
| Sublime Text | macOS, Windows, Linux |
| Atom | macOS, Windows, Linux |
| Zed | macOS |
| Nova | macOS |

## Set Default IDE

1. Open **Settings** (gear icon)
2. Select your IDE under **Default IDE**
3. Close settings

Now clicking the IDE button opens projects directly.

## Open in IDE

Click the **code icon** on any project card.

If no default is set, you'll be prompted to choose.

## Terminal Integration

### Supported Terminals

| Terminal | Platforms |
|----------|-----------|
| Terminal.app | macOS |
| iTerm | macOS |
| Hyper | macOS, Windows, Linux |
| Warp | macOS |
| Kitty | macOS, Linux |
| Alacritty | macOS, Windows, Linux |
| PowerShell | Windows |
| Windows Terminal | Windows |
| CMD | Windows |
| GNOME Terminal | Linux |
| Konsole | Linux |

### Set Default Terminal

1. Open **Settings**
2. Select your terminal under **Default Terminal**

### Open in Terminal

Click the **terminal icon** on any project card.

The terminal opens in the project directory, ready to run commands.
