# Getting Started

Welcome to Tafil! This guide will help you get up and running quickly.

## What is Tafil?

Tafil is a desktop application for managing Node.js projects. It helps you:

- **Scan** directories to find all your Node.js projects
- **Organize** projects into collections
- **Launch** dev servers with one click
- **Open** projects in your favorite IDE

## Installation

Download Tafil for your platform:

- [macOS](/downloads/macos)
- [Windows](/downloads/windows)
- [Linux](/downloads/linux)

## Quick Start

### 1. Launch Tafil

Open Tafil from your Applications folder (macOS), Start Menu (Windows), or app launcher (Linux).

### 2. Scan Your Projects

Click **Scan Home** to find all Node.js projects in your home directory.

Or use **Scan Folder** to scan a specific directory.

### 3. Run a Project

Click the **Play** button on any project card to start its dev server.

### 4. Open in IDE

Click the **Code** icon to open the project in your default IDE.
