# First Steps

After installing Tafil, follow these steps to set up your workflow.

## 1. Set Default IDE

1. Open **Settings** (gear icon in sidebar)
2. Under **Default IDE**, select your preferred editor:
   - VS Code
   - Cursor
   - WebStorm
   - Sublime Text
   - IntelliJ IDEA
   - Atom

This lets you open projects with one click instead of choosing each time.

## 2. Set Default Terminal

1. In Settings, under **Default Terminal**, select:
   - **macOS:** Terminal, iTerm, Hyper, Warp, Kitty, Alacritty
   - **Windows:** PowerShell, Windows Terminal, CMD
   - **Linux:** GNOME Terminal, Konsole, Alacritty

## 3. Scan Your Projects

Click **Scan Home** to find all Node.js projects in your home directory.

For better performance, use **Scan Folder** on specific directories like:
- `~/projects`
- `~/code`
- `~/work`

## 4. Create Collections

Organize projects by clicking **+ New Collection** in the sidebar.

Examples:
- "Work" - Client projects
- "Personal" - Side projects
- "Learning" - Tutorial projects

## 5. Add Projects to Collections

Click the **folder icon** on any project card to add it to a collection.

## Interface Overview

| Element | Description |
|---------|-------------|
| Sidebar | Navigate collections and views |
| Project Cards | Show project info and actions |
| Running Badge | Indicates active dev server |
| Port Number | Shows which port is in use |
